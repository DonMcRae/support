<h1>Gnip Support</h1>
<h2>Example Code Snippets</h2>
<p>This repository contains sample code snippets that demonstrate the basic API methods available on Gnip's API.  These are broken into two categories:
	<ul>
		<li>
			Premium Stream Connection - Establish a streaming connection to a premium Gnip stream</li>
		<li>
			Rules API - Add, Delete, and List rules on premium and Enterprise Data Collector streams</li>
	</ul>
</p>
<p>The snippets are available in the following programming languages:
	<ul>
		<li>
			C#</li>
		<li>
			Java</li>
		<li>
			PHP</li>
		<li>
			Python</li>
		<li>
			Ruby</li>
	</ul>
</p>
